# SmileAxis Orthodontics

Welcome to the **SmileAxis Orthodontics** website repository! This is a modern, responsive website built for an orthodontic practice in the Austin area, designed to provide a premium patient experience.

## Homepage Overview

<img src="assets/my%20images/homepage%20screenshot.png" alt="SmileAxis Homepage Screenshot" width="100%">

## Features

- **Modern & Responsive Design:** Beautifully designed pages that look great on any device (desktop, tablet, and mobile).
- **Interactive Booking Form:** A step-by-step modal designed for easy and intuitive appointment scheduling.
- **Premium User Experience:** High-quality unique imagery, clean typography, smooth interactions, and clear navigation.
- **Comprehensive Sections:** Includes detailed pages for Treatments, Results, Reviews, Financing, and Locations.

## Technologies Used

- HTML5
- CSS3 (Custom Design System, Flexbox, CSS Grid)
- JavaScript (Vanilla JS for interactivity, FAQs, and modals)

## Project Structure

- `index.html`: The main homepage.
- `treatments.html`: Detailed information about the treatments offered (Invisalign, Traditional Braces, etc.).
- `results.html`: Smile gallery and before/after showcasing successful treatments.
- `reviews.html`: Patient reviews and testimonials.
- `financing.html`: Payment options, insurance, and all-in pricing plans.
- `locations.html`: Clinic details, hours, and direction for multiple Austin-area clinics.
- `styles.css`: Main stylesheet containing the design system and UI components.
- `script.js`: Interactive logic for elements like the booking modal, FAQ accordions, and mobile navigation.
- `assets/`: Contains all images (including unique photography), logos, and icons.
